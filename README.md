# Cyber Technology Reference Architecture

A reference architecture for cybersecurity technology, covering security controls, frameworks, and implementation patterns.

## Purpose

This repository documents a structured approach to cybersecurity architecture — mapping technologies to security domains, frameworks, and threat models.

## Structure

```
cyber-tech-ref-architecture/
├── architecture/        # Architecture diagrams and descriptions
├── controls/            # Security controls by domain
├── frameworks/          # Framework mappings (NIST CSF, ISO 27001, etc.)
├── technologies/        # Technology evaluations and patterns
└── templates/           # Configuration and deployment templates
```

## Frameworks Referenced

- NIST Cybersecurity Framework (CSF 2.0)
- NIST SP 800-53
- ISO/IEC 27001
- MITRE ATT&CK
- CIS Controls
